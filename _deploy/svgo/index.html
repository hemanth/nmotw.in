svgo

SVG Optimizer

Optimizes SVG by cleaning:

  • Editor metadata.

  • Comments.

  • Hidden elements.

  • Default or non-optimal values.

  • Other stuff that can be safely removed or converted without affecting SVG rendering result.

Get it: npm install --global svgo
