printf

Get it: npm install printf

Sample usage:

Flags:

  • (space)

  • +

  • -

  • 0

Features:

  • Width / precision => %2f

  • Numerical bases => %c

  • Miscellaneous => %d%%, +%s%+

  • Extra => The %O converter will call util.inspect(...) at the argument, %-*.*f.
