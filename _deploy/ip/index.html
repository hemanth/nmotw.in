ip

IP address utilities.

ip sweet and simple module that provide the below util methods to play with IP!

  • toBuffer
  • toString
  • isV4Format
  • isV6Format
  • fromPrefixLen
  • mask
  • cidr
  • subnet
  • cidrSubnet
  • not
  • or
  • isEqual
  • isPrivate
  • isPublic
  • isLoopback
  • loopback
  • address
  • toLong
  • fromLong

Get it: npm install --save ip
